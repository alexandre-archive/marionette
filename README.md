# marionette

Marionette game using Leap Motion Controller

## Requirements

### Hardware

- LeapMotion

### Backend

- Node.JS
- Bower
- Gulp

### Frontend

- AngularJS
- Bootstrap
- jQuery
- Klass
- LeapJS
- Underscore

## Setup

Before all, you need to install [Node.JS](https://nodejs.org/).

To install all dependencies run:

`npm install`

To download all third part libs run:

`bower install`

And that's it.
